# FolderFlow

FolderFlow は Mac OS Finder のカラム表示やリスト表示でデータをツリーで確認できる簡易ファイルマネージャーです。

## 特徴

- Finder風カラム表示
- フォルダカラータグ
- フォルダーのツリー確認
- データ容量の MB/GB/TB 表記
- 2ペイン表示
- アイコン表示（Ctrl＋スクロールで拡大・縮小）
- ライト/ダークモード切替
- タブ機能
- RAWファイルプレビュー
- 高速プレビュー

## 動作環境

- Windows 11
- 64bit
